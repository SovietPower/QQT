**一些难点：**

所有图片的大小、图像输出的偏移量（精确到1像素），都要自己对比游戏原图调。。要保证所有元素图片的每一条棱，与游戏中对应元素对齐。

所有图片都是抠的，每种类型的好多图片都要做一致的处理。如果有种类型的一点点参数不满意，需要全部重改。
